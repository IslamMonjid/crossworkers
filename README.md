## Prerequisite:
- Composer
- Symfony CLI
- MySQL
- PHP >= 8.1

## To Install Dependencies

Please run the following command in the project directory: 

- composer install

## To Migrate Database

Please run the following command in the project directory: 

- php bin/console doctrine:migrations:migrate


## To Run The Apllication 

- symfony server:start
